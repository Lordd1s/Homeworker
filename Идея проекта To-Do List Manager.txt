Идея проекта: To-Do List Manager - Управление списком задач

1) Идея проекта:

To-Do List Manager - это веб-приложение, созданное с помощью Django MVT, которое позволяет пользователям создавать, отслеживать и управлять своими списками задач. Проект предоставляет простой и интуитивный интерфейс для управления ежедневными задачами.

2) Описание логики:

Модели:

User: Основная информация о пользователях.
Task: Задачи пользователя с полями, такими как "Заголовок", "Описание", "Дата создания", "Дата выполнения" и "Статус выполнения".
Представления (Views):

Главная страница: Показать приветственное сообщение и ссылки на вход или регистрацию.
Страница входа: Обработка входа пользователя с использованием формы.
Страница регистрации: Обработка регистрации нового пользователя с использованием формы.
Панель управления задачами: Отображение списка задач пользователя, возможность создания новых задач, отметки задач как выполненных и удаления задач.

Шаблоны (Templates):

HTML-шаблоны для каждой страницы, используя базовый HTML и вставки переменных для динамической информации.
URL-Маршрутизация (URL Routing):

Настройка URL-маршрутов для каждой страницы и связь с соответствующими представлениями.
Логика форм (Forms):

Форма для входа пользователя с полями "Имя пользователя" и "Пароль".
Форма для регистрации пользователя с полями "Имя пользователя", "Пароль" и "Подтверждение пароля".
Форма для создания и управления задачами.
Аутентификация и Авторизация:

Обработка аутентификации пользователей при входе.
Ограничение доступа к панели управления задачами только для авторизованных пользователей.
Бизнес-логика:

Возможность создания, редактирования и удаления задач.
Отметка задач как выполненных.
Сортировка и фильтрация задач по статусу выполнения и датам.
